Ping Pong exercise using a Javascript function

Created by CiaoChow

The program is different from your standard Fizzbuzz exercise. Instead, the user is prompted to supply an whole number of their choice. We then verify that the user input is a real number (Ex: 12, not 'twelve') and that it is a whole number (no decimal points). We then count from 1 to the user's number and display the results. It will replace numbers divisible by both 3 and 5 with “ping-pong”, those divisible by 3 with “ping”, and those divisible by 5 with “pong”.
